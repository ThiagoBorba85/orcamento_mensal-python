# 📊 Analisador de Orçamento Mensal

Este projeto é um script em **Python** desenvolvido para facilitar a organização financeira pessoal. Ele coleta dados de renda e despesas diretamente pelo terminal e automatiza a geração de um relatório detalhado em **Excel**.

## 🚀 Funcionalidades
- **Entrada Dinâmica:** Aceita valores no formato brasileiro (ex: 4.350,00 ou 4350).
- **Integração com Excel:** Gera uma planilha organizada com abas de "Gastos" e "Resumo".
- **Tratamento de Erros:** Validação de entradas para evitar fechamentos inesperados.

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
- **Pandas:** Para estruturação e manipulação dos dados (DataFrames).
- **OpenPyXL:** Para a exportação e criação dos arquivos .xlsx.

## 📁 Estrutura do Projeto
- `orcamento_mensal.py`: Código-fonte principal.
- `.gitignore`: Configurado para proteger dados sensíveis (não envia a planilha pessoal para o GitHub).

2. Estruturas de repetição e dicionários.
